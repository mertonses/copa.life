# copa.life SVG refresh pack

Bu pakette şunlar var:
- 12 adet howto ikonu → `howto/`
- 4 adet shout butonu ikonu → `shout/`
- 5 adet kart trait ikonu → `traits/`
- 5 adet rakip karakter / OPPCHARS ikonu → `oppchars/`
- 1 adet MOTM yıldızı → `motm/`
- loader renk düzeltmesi → `snippets/loader_color_fix.txt`

## Teknik notlar
- Howto ikonları: `32x32`, `stroke-width="1.6"`
- Shout ikonları: `24x24`, `stroke-width="1.8"`, fill/stroke `currentColor`
- Trait / OPP icons: `20x20`, `stroke-width="1.6"`
- MOTM yıldızı: `16x16`, `stroke-width="1.5"`
- Tüm ikonlar square viewbox ile çizildi.
- Shout ikonlarında lit state için hard-coded renk kullanılmadı.
