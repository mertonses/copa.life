# Audio Asset Slots

This folder is reserved for match audio assets used by `MatchAudioDirector`.

Recommended first pack:

- `crowd/idle`, `crowd/active`, `crowd/danger`
- `crowd/reactions/goal`, `miss`, `save`, `disappointed`
- `field/short_pass`, `hard_pass`, `long_ball`, `cross`, `shot`
- `field/tackle`, `keeper_save`, `net_hit`, `post_hit`, `whistle`
- `voices/general`, `voices/keeper`, `voices/reactions`

Use only assets that are safe for commercial release, preferably CC0, MIT, original, or explicitly licensed for commercial use. Keep attribution notes next to imported sounds.
